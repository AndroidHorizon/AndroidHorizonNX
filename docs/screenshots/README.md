# Screenshots

The app captures these automatically on real hardware into
`sdmc:/AndroidHorizonNX/screenshots/`:

| File | When it's taken |
|------|-----------------|
| `ui_menu.png` | APK browser, ~3 s after launch (animations settled) |
| `ui_loading.png` | Loading screen at ≥40 % progress (live log visible) |
| `ui_result.png` | Launch result screen, first frame |
| `ui_about.png` | About screen, once the GitHub avatar has loaded |
| `game_frame30.png` | Game GL framebuffer at frame 30 (early splash) |
| `game_frame300.png` | Game GL framebuffer at frame 300 (post-splash) |
| `game_frame900.png` | Game GL framebuffer at frame 900 (menu, if reached) |

After a test run, copy the PNGs from the SD card into this folder — the main
README embeds them from here.
